# 🏥 Base de Données PEV RDC

Base de données PostgreSQL pour le Programme Élargi de Vaccination en RDC, conforme au modèle Power BI.

## 🚀 Installation Rapide

### Prérequis
- Docker Desktop installé
- Python 3.8+
- 4GB RAM minimum

### Installation
```bash
# 1. Clonez le projet
git clone <votre-repo>
cd pev-rdc-database

# 2. Installez les dépendances Python
pip install -r requirements.txt

# 3. Lancez l'installation complète
python import_data.py
